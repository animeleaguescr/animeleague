## ⚙️ Configuration

The Anime League Script ships with a flexible configuration file that you can edit before or after injection. Below is the default `config.lua` layout:

| Setting | Default Value | Description |
|---------|--------------|-------------|
| `AimbotEnabled` | `true` | Master toggle for aimbot module |
| `AimbotFOV` | `120` | Field-of-view cone in pixels |
| `AimbotSmoothing` | `0.4` | Crosshair travel speed — lower is snappier |
| `ESPEnabled` | `true` | Toggle full ESP wallhack overlay |
| `ESPShowHealth` | `true` | Display enemy health bars |
| `ESPShowDistance` | `true` | Show distance in studs |
| `GunModFireRate` | `1.5x` | Fire rate multiplier |
| `SpeedMultiplier` | `1.3x` | Movement speed factor |
| `AutoFarmEnabled` | `false` | Toggle AFK auto-farm |
| `SilentAimEnabled` | `false` | Server-side aim correction |

```bash
-- Anime League Script / Default Configuration
-- File: C:/Users/YourName/Documents/AnimeLeagueScript/config.lua

local Config = {
    -- Aimbot Settings
    Aimbot = {
        Enabled       = true,
        FOV           = 120,
        Smoothing     = 0.4,
        TargetBone    = "Head",    -- Options: Head, Torso, Closest
        TeamCheck     = true,
        VisibleCheck  = true,
        Keybind       = "RightMouse",
    },

    -- ESP Settings
    ESP = {
        Enabled       = true,
        ShowBoxes     = true,
        ShowHealth    = true,
        ShowDistance   = true,
        ShowNames     = true,
        RenderDistance = 800,       -- max studs
        TeamColor     = Color3.fromRGB(0, 255, 128),
        EnemyColor    = Color3.fromRGB(255, 60, 60),
    },

    -- Gun Modification
    GunMod = {
        FireRateMulti = 1.5,
        SpreadOverride = 0,
        RecoilOverride = 0,
        DamageMulti    = 1.0,
        InfiniteAmmo   = true,
    },

    -- Movement
    Movement = {
        SpeedMulti    = 1.3,
        InfiniteJump  = true,
        NoClip        = false,
    },

    -- Auto Farm
    AutoFarm = {
        Enabled       = false,
        CollectCrates = true,
        AutoObjective = true,
        AntiAFK       = true,
        FarmDelay     = 0.5,       -- seconds between actions
    },
}

return Config
```

## 🛠️ Troubleshooting Common Issues

**❌ Script GUI does not appear after execution**
The overlay may fail to render if your executor does not support `Drawing` library calls or if Roblox updated its rendering pipeline. **Solution:** Update your executor to the latest version, re-download the Anime League Script, and try injecting after the game fully loads past the loading screen.

**❌ Aimbot locks onto teammates**
This happens when `TeamCheck` is set to `false` in the configuration file. **Solution:** Open `config.lua` and set `Config.Aimbot.TeamCheck = true` so the Anime League Script ignores friendly players.

**❌ FPS drops when ESP is enabled**
Large servers with 30+ players can stress the ESP renderer, especially on integrated graphics. **Solution:** Lower `Config.ESP.RenderDistance` to `400` studs or disable `ShowBoxes` to reduce draw calls within the Anime League Script.

**❌ Auto Farm stops collecting crates**
A recent map update may have shifted crate spawn positions. **Solution:** Update to the newest Anime League Script version where spawn coordinates are patched, or toggle `CollectCrates` off and on to force a position refresh.

**❌ Script throws an error on mobile**
Mobile executors occasionally have Lua compatibility gaps. **Solution:** Make sure you are using a mobile executor that supports `getgenv()` and `hookfunction()`. Reinstall the Anime League Script if the error persists after updating.
